PyPulse v0.0.1
==============

PyPulse is a package used to handle PSRFITS files and perform subsequent analyses on pulse profiles.

Download:

`Github <https://github.com/mtlam/PyPulse>`_

Requirements:

* `python <https://www.python.org/>`_ 2.7 or 3.X 
* `numpy <http://www.numpy.org/>`_
* `scipy <http://www.scipy.org/>`_
* `matplotlib <http://matplotlib.org/>`_
* `astropy <http://www.astropy.org/>`_ or `pyfits <https://pythonhosted.org/pyfits/>`_
